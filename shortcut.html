IntelliJ IDEA 단축키 15가지



Alt + Enter

코드에서 오류가 발견된 경우 해당 오류 위에 커서를 놓고 Alt+Enter를 누르면 문제에 대한 수정 제안 목록이 표시된다.

Alt+Enter를 사용하여, 수정 제안 중 하나를 선택할 수 있다.



F2

F2를 사용하여 다음 오류, 경고 또는 수정 제안으로 점프할 수 있다.

F2와 Alt+Enter를 함께 사용하여 모든 제안을 표시하고 하나를 선택하거나,
Shift+Alt+Enter를 사용해 첫 번째 제안을 적용할 수 있다.



⌘1 또는 Alt+1

⌘1(MacOS) 또는 Alt+1(Windows/Linux)를 사용하면 프로젝트 창이 열리고 해당 위치에 포커스가 위치한다.

방향 키를 사용해 트리를 탐색하거나 입력하여 검색할 수 있다.



Esc

열린 도구 창과 상관 없이 에디터에 다시 포커스가 위치하게 되어 코드 작업을 계속 수행할 수 있다.

Esc는 변경 사항을 적용하지 않고 팝업 창을 닫을 때도 유용하다.



⌘E 또는 Ctrl+E

⌘E(MacOS) 또는 Ctrl+E(Windows/Linux)를 사용해 최근 실행했던 파일을 확인할 수 있다.

IntelliJ IDEA의 다른 창에서처럼 특정 항목을 검색할 수 있다.



⌘B 또는 Ctrl+B

⌘B(MacOS) 또는 Ctrl+B(Windows/Linux)를 사용하면 심볼의 선언으로 이동할 수 있다.

⌥⌘B(MacOS) 또는 Ctrl+Alt+B(Windows/Linux)를 누르면 구현으로 이동할 수 있다.



Alt+F7

Alt+F7을 누르면 어떤 항목이 사용된 모든 곳이 표시된다.

인터페이스 이름 위에서 Alt+F7을 누르면 필드 선언/클래스 상관 없이
해당 인터페이스가 사용된 모든 곳이 검색 창에 표시된다.



Ctrl Ctrl

IDE 안에 있는지의 여부나 열린 파일과 상관 없이 Ctrl을 두 번 누르면 Run Anything 창이 열린다.

기본적으로 최근 실행 구성 목록이 표시되지만, 항목의 이름을 입력하여 다른 실행 구성을 검색하고 실행할 수도 있다.



⌥↑ / ⌥↓ 또는 Ctrl + W / Shift + Ctrl + W

커서 근처의 코드 선택 영역을 확대하거나 축소할 수 있다.
선택 영역을 확대할 경우 IntelliJ IDEA가 확대된 영역에서 유효한 다음 식을 자동으로 선택한다.

⌥↓(MacOS) 또는 Ctrl+Shift+W(Windows/Linux)를 사용하면 선택 영역이 다시 커서 위치로 축소된다.



⌘/ 또는 Ctrl + /

줄 위의 아무 곳에서나 ⌘/(MacOS) 또는 Ctrl+/(Windows/Linux)를 누르면 해당 코드 줄이 줄 주석으로 처리된다.
이미 주석으로 처리된 경우 동일한 단축키를 누르면 해당 줄의 주석 처리가 제거된다.

전체 코드 블록을 선택하면 ⌥⌘/(MacOS) 또는 Shift+Ctrl+/(Windows/Linux)를 사용해 블록 주석을 추가할 수 있다.
코드 블록 안의 아무 곳에서나 커서를 사용해 이 단축키를 다시 누르면 블록 주석이 제거된다.



⇧⌘⏎ 또는 Shift+Ctrl+Enter

코딩을 할 때 현재 구문 완성을 자주 사용한다면 대부분의 경우 단순히 코드 끝에 세미콜론을 추가하는 것이다.
예를 들어, “for” 루프를 작성하는 중 이 단축키를 누르면 IntelliJ IDEA가 중괄호를 추가하고 블록 안에 커서를 배치한다.

IDE가 문법을 완성하기 위해 더 이상 코드를 추가할 필요가 없을 경우에도 이 단축키를 사용해 적당한 다음 위치에 커서를 배치할 수 있어서 매우 유용하다.



⌥⌘L 또는 Ctrl+Alt+L

해당 프로젝트의 표준에 맞도록 현재 파일의 서식을 지정할 수 있다.

다른 범위를 선택하여 서식을 지정하고 싶으면 ⇧⌥⌘L(MacOS) 또는 Shift+Ctrl+Alt+L(Windows/Linux)을 사용하여 전체 파일의 서식을 다시 지정하도록 선택할 수 있다.



⌃T 또는 Shift+Ctrl+Alt+T

심볼 또는 선택 영역 위에서 이 단축키를 누르면 사용 가능한 리팩토링 옵션이 표시된다.

특정 리팩토링의 키보드 단축키가 존재할 경우 대화 상자에도 표시되어 다음 번에 바로 사용할 수 있다.



⇧⌘A 또는 Shift+Ctrl+A

IntelliJ IDEA에서 모든 액션을 검색할 수 있다.
드롭다운 메뉴에 액션뿐만 아니라 단축키도 표시되므로 이 단축키를 배우고 연습할 수 있다.

바로 설정을 변경할 수 있으며, 도구 창을 검색하고 열 수 있다.



Shift shift

Shift 키를 두 번 누르면 모든 항목을 검색할 수 있는 검색 상자가 열리며 액션 검색처럼 이를 사용해 설정을 변경할 수 있다.

검색할 항목을 입력하면 클래스, 파일, 심볼 및 액션에 대한 결과가 표시된다.
전체 검색은 명령어를 지원하므로 에디터에 대한 설정을 검색할 수 있다.
